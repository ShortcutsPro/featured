Reflect on the day This shortcut asks you five questions to review your working day and set goals for the next day. It stores answers in Notes to review how you have been doing across a period of time. The last two questions set reminders for the next day. Use with Siri to have a conversation. "Hey Siri, Reflect on the day" or set as an automation to run at the end of each day. ---
